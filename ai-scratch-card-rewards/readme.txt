== Description ==

**AI Scratch Card Rewards** allows you to create customizable, interactive scratch cards on your WordPress site. Users can scratch to reveal prizes, capture leads, and boost engagement ‚Äî all while staying GDPR-compliant.

Perfect for giveaways, email capture, loyalty campaigns, and gamified promotions.

- Gutenberg block + shortcode support

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin via the **Plugins** screen in WordPress
3. Go to **Scratch Cards ‚Üí Create New** to build your first card
4. Copy the generated shortcode and paste it into any page or post
5. (Optional) Enable lead capture or webhook integrations from the card settings

== Usage ==

Use the shortcode:

`[ai_scratch_card id="123"]`

To display the scratch card with ID 123.
